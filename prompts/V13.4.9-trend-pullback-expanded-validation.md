# 12. Slippage-Adjusted Metrics

V13.4.9 必须加入滑点后处理指标。

默认滑点：

```text
slippageRate = 0.0005
即 0.05% one-way
```

对每笔交易：

```text
roundTripSlippage = slippageRate * 2
```

如果使用 profit ratio：

```text
slippageAdjustedProfitRatio = rawProfitRatio - slippageRate * 2
```

如果使用 USDT profit：

```text
slippageCost = abs(positionNotional) * slippageRate * 2
slippageAdjustedProfit = rawProfit - slippageCost
```

如果无法准确获得 positionNotional：

```text
slippageCostEstimated = true
```

报告必须明确：

```text
slippageAppliedByFreqtrade = false
slippageAppliedByPostProcessing = true
```

不要把后处理滑点伪装成 Freqtrade 原生滑点。

V13.4.9 不批准 Dry-run，但要判断是否值得进入下一步。

建议 Gate：

```text
1. slippageAdjustedTotalReturnPct > 0
```
