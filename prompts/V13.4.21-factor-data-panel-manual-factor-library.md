# AlphaPilot V13.4.21 - FactorDataPanel and Manual Factor Library Implementation

## 给 Codex 的执行说明

请严格按照本文档执行 V13.4.21。

本版本的目标是：**基于 V13.4.20 设计好的 Alpha Factor Research Layer，实现 FactorDataPanel 与 Manual Factor Library 的本地数据生成层，让 AlphaPilot 能从本地公开 OHLCV 数据生成研究用因子面板，为后续因子评价和 Strategy Research Factory 做准备。**

本版本仍然是 research-only。  
本版本不是交易策略实现版。  
本版本不是策略回测版。  
本版本不是 Dry-run 版。  
本版本不是实盘版。  
本版本不接真实 API Key。  
本版本不自动交易。  
本版本不创建真实订单。  

完成后请输出完整自检报告。

# 10. FactorDataPanel Builder

新增：

```text
alphapilot/factors/factor_data_panel.py
```

功能：

```text
1. 读取本地 OHLCV
2. 转成统一 panel rows
3. 计算基础收益字段
4. 估算 quoteVolume = close * volume
5. 计算 vwap fallback
6. 关联 Dynamic Universe membership，如 snapshots 可用
7. 输出 factor panel sample
8. 输出 build report
```

注意：

```text
quoteVolume 如果是估算，必须记录 quoteVolumeEstimated=true
vwap 如果无法从真实成交额计算，可用 typical price fallback，并记录 vwapEstimated=true
```

# 14. Factor Computation Engine

新增：

```text
alphapilot/factors/compute_manual_factors.py
```

功能：

```text
1. 输入 factor panel
2. 按 pair 分组计算时间序列因子
3. 按 timestamp 横截面计算 rank / zscore 因子
4. 输出 factor columns
5. 输出 factor computation report
```

要求：

```text
不得使用未来数据
rolling 只能使用当前及历史窗口
cross-sectional rank 只能使用同一 timestamp 的横截面数据
```
