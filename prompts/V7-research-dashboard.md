# AlphaPilot V7 - Research Dashboard

Status: Draft / To Be Expanded

## Goal

Create a research dashboard that helps traders inspect their behavior, market
features, and strategy patterns without encouraging impulsive trading.

## Scope

- Add research-first dashboard views
- Show trade pattern summaries
- Show feature quality and coverage
- Show discipline and execution metrics
- Support filtering by symbol, setup, direction, and time

## Non Goals

- No trading terminal
- No live order panel
- No signal feed
- No price prediction
- No automatic trading

## Safety Rules

- Dashboard must prioritize evidence over opinion
- Dashboard must not recommend buys or sells
- Dashboard must distinguish correlation from causation
- Dashboard must show data quality limits
- Dashboard must keep risk context visible

## Acceptance Criteria

- Research views are defined
- Metrics are based on stored data
- Data quality is visible
- UI remains calm and non-stimulating
- README or docs describe the dashboard purpose
