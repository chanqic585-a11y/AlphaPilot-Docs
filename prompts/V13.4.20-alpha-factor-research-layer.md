# 17. Benchmark Suite Purpose

Benchmark 用于回答：

```text
我们的复杂策略是否真的比简单策略好？
是否比不交易好？
是否比持有 BTC 好？
是否比简单 EMA 趋势好？
是否比简单 RSI 回归好？
```

比较指标：

```text
total return
slippage-adjusted return
max drawdown
profit factor
win rate
trade count
max loss streak
monthly stability
pair stability
```
