# 10. Failure Review Requirements

新增失败复盘必须覆盖以下问题。

## 10.1 交易频率问题

根据 V13.4.5：

```text
Top30 6个月 trades：
V01 2701
B 1515
C 2496
E 2671
```

必须分析：

```text
1. 交易次数过多是否导致手续费吞噬
2. 15m 高频信号是否过噪
3. 是否需要更少、更高质量信号
```

---

## 10.2 成本问题

根据 V13.4.5：

```text
slippage-adjusted 后所有候选更差
```

必须分析：

```text
1. 策略是否成本敏感
2. 15m 策略是否不适合当前信号密度
3. V03 是否必须减少交易频率或提高目标收益/盈亏比
```

---

## 10.3 Payoff 结构问题

当前策略：

```text
-3% stoploss
+3% take profit
```

在胜率约 41% 情况下难以盈利。

必须分析：

```text
1. 1:1 payoff 在当前胜率下不够
2. 扣除手续费和滑点后需要更高胜率或更高 reward/risk
3. V03 是否需要至少 1.5R 或 2R 目标
```

---

## 10.4 入场质量问题

当前策略在扩大验证中深度亏损。

必须分析：

```text
1. 当前反弹信号质量不足
2. volumeRatio / RSI / MACD / EMA20 组合不能有效识别正期望入场
3. V03 应引入更强结构确认
```

---

## 10.5 趋势过滤问题

V13.4.2 显示：

```text
weak_4h_trend 是最大 skip reason
```

但通过过滤的信号仍然整体亏损。

必须分析：

```text
1. 当前 4h 趋势过滤有拦截作用
2. 但不是充分条件
3. V03 不能只依靠 4h EMA 过滤
```

---

## 10.6 SOL / Pair Risk 问题

V13.4.1 / V13.4.2 显示 SOL 贡献亏损较大。

必须分析：

```text
1. 单 pair 风险可能拖累组合
2. 不应基于单次 smoke 永久排除 SOL
3. V03 应加入 pair-level exposure / signal cap / risk watchlist
```

# 13. V03 Candidate Directions

至少提出 3 个 V03 方向。

---

## 13.1 V03A - Trend Pullback Continuation

定位：

```text
趋势回调延续策略
```

核心思想：

```text
只在更明确的上升趋势中，等待价格回调到结构支撑 / EMA 区域，并出现重新走强确认。
```

建议规则方向：

```text
1. 4h 趋势明确向上
2. 1h 趋势不破坏
3. 15m 回调到 EMA20 / EMA50 / 支撑区域
4. 出现放量重新站上
5. 不在强空头中做反弹
```

优点：

```text
比 V0.1 的“弱势反弹”更稳
减少接飞刀
```

风险：

```text
信号少
可能错过早期反转
```

---

## 13.2 V03B - Breakout Retest Confirmation

定位：

```text
突破回踩确认策略
```

核心思想：

```text
不做随机反弹，而是等待价格突破关键压力后回踩确认，再进入。
```

建议规则方向：

```text
1. 识别近期阻力
2. 15m / 1h 收盘突破
3. 回踩不跌回阻力下方
4. 成交量确认
5. BTC 环境安全
```

优点：

```text
入场结构更清晰
可解释性更强
```

风险：

```text
假突破
回踩不出现
```

---

## 13.3 V03C - High Score Signal Only

定位：

```text
低频高质量信号策略
```

核心思想：

```text
保留放量反弹思路，但引入更严格的人工多因子评分，只有高分信号进入回测。
```

建议因子：

```text
大盘安全
趋势环境
反弹位置
成交量质量
动能改善
风险质量
```

要求：

```text
score >= 80 才入场
```

优点：

```text
明显降低交易频率
可逐步接入机器学习
```

风险：

```text
信号过少
权重需要谨慎验证
```

---

## 13.4 Optional V03D - 1h Main Timeframe

定位：

```text
降低噪音的 1h 策略
```

核心思想：

```text
将主周期从 15m 提高到 1h，减少噪音和手续费消耗。
```

优点：

```text
交易频率降低
成本影响降低
信号更稳定
```

风险：

```text
入场滞后
样本更少
```

# 14. V03 Quality Gate

V03 进入 Dry-run 前，必须至少满足：

```text
1. slippage-adjusted total return > 0
2. slippage-adjusted profit factor > 1.15
3. max drawdown 明显低于 V0.1
4. max consecutive losses 可接受
5. trade count 足够但不过度
6. 没有单一 pair 极端主导收益或亏损
7. 至少通过 smoke + 6个月 Top30
8. 最好通过更长 timerange
```

更严格目标：

```text
profit factor >= 1.2
max drawdown < 25%
trade count 适中
slippage 后仍为正
```

如果达不到：

```text
不得进入 Dry-run
```
